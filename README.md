# HOUSE OF THE TIMER 

## Simple timers (time since), configurable on react FE and saved to Python BE


## Installation:

### frontend
1) ```cd frontend```
2) ```npm install```
3) ```npm start ```

### backend
1) ```cd backend```
2) ```uvicorn main:app --host 0.0.0.0 --port 5000 --reload ```
